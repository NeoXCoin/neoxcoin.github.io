🛒 如何購買 NeoX？

只需幾個簡單步驟，即可購買 NeoX Coin (NXC)：

1️⃣
安裝並開啟 🦊MetaMask 錢包
👉 前往官方網站下載
2️⃣
把 Base 主網 加入到你的錢包
👉 使用 Chainlist.org 一鍵加入 Base
3️⃣
將 ETH 橋接到 Base 網絡
👉 先把 ETH 轉入 Ethereum 主網的 MetaMask
👉 然後使用 官方 Base Bridge 把 ETH 轉換為 Base ETH
⚠️ 購買 NXC 及支付 Gas 費需要 Base 網絡上的 ETH。
4️⃣
點擊下方按鈕，在官方平台購買 NeoX Coin
