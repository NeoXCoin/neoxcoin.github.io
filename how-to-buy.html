How to Buy NeoX

1. Connect your wallet (e.g. MetaMask) to the Sepolia Testnet.

2. Make sure you have test ETH. You can get it from a faucet.

3. Visit the official NeoX Token page on Thirdweb.

4. Click “Buy” and follow on-screen instructions.

5. Optional: Add NXC token to MetaMask using the button below.
