概覽（Overview）

NeoX 是建基於 Base Mainnet 的去中心化 Web3 生態，結合數碼藝術、區塊鏈遊戲以及元宇宙式互動體驗，並以同一品牌呈現。 生態當中包括 NeoX Coin（NXC）代幣、策展式 NFT 系列、HTML5 街機遊戲， 以及 AI 強化的多媒體項目。

起源與發展歷史（History）

NeoX 最初以創意數碼藝術項目形式出現，之後逐步加入區塊鏈技術、NFT 和互動媒體元素。到 2025 年，NeoX Coin（NXC）在 Base Mainnet 上正式發行， 標誌著 NeoX 從單一藝術計劃，演變為涵蓋收藏品、遊戲及元宇宙實驗的 更廣泛鏈上生態。

NeoX Coin（NXC）代幣

NXC 是 NeoX 生態中的實用代幣（utility token）， 於 Base Mainnet 上部署，用作支援 NeoX 各種體驗，以及未來元宇宙功能。

NFT 系列（NFT Collections）

NeoX 目前包含多個精選 NFT 系列，例如：

  • NeoX Timekeeper Edition – 以「時間與感知」為主題的 8 枚限量 NFT。
  • NeoX Van Gogh Visions – 結合梵高畫風與 NeoX 美學的藝術創作。
  • NeoX Abstract Realms – 側重概念與抽象表達的藝術系列。
  • NeoCoin Genesis Edition（G1） – NeoX 早期的重要基礎藝術系列。

NeoX Catch 遊戲

NeoX Catch 是一款以瀏覽器運行的 HTML5 街機式遊戲，玩家需要接住 NeoX 金幣、避開障礙物，並在不同關卡中提升分數。

主要特色包括：

  • 多關卡設計，配有開場動畫與結尾畫面
  • 分數記錄與 Top 3 排行榜展示
  • 以手機和平板為優先的響應式版面
  • 整體視覺與 NeoX 品牌風格一致

元宇宙展覽（Metaverse Exhibition）

NeoX 亦在 OnCyber 等平台維持虛擬展覽空間，展示 NFT 作品、3D 虛擬角色、 氣氛燈光與電影式場景。隨著新系列與實驗空間推出，展覽內容會持續更新。

技術棧（Technology Stack）

NeoX 使用的主要技術包括：

  • Base Mainnet（二層）作為代幣及合約部署網絡
  • Thirdweb 用於智能合約部署與管理
  • IPFS / Lighthouse 作為去中心化檔案儲存
  • HTML5 ＋ Canvas 實作遊戲機制與動畫
  • VRM / glTF 資產，用於虛擬角色與元宇宙視覺
  • 針對手機及 iOS 裝置優化的響應式設計

未來發展（Future Development）

未來計劃包括：擴展更多元宇宙場景、新 NFT 系列、AI 輔助微電影項目、 進一步升級遊戲玩法，以及加強 Web、Mobile、虛擬世界之間的跨平台整合。
