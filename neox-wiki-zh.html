NeoX 官方生態百科（Wiki）

項目概述

NeoX 是一個建立於 Base Mainnet 的分散式 Web3 生態系統， 結合數碼藝術、區塊鏈遊戲和元宇宙創意，形成一套完整的品牌宇宙。 生態包含 NeoX Coin (NXC)、NFT 系列、網頁遊戲、AI 視覺作品與虛擬展館。

發展歷史

NeoX 最初源自藝術創作與數碼收藏，隨後逐步擴展至區塊鏈技術領域。 在 2025 年，NeoX Coin (NXC) 正式於 Base Mainnet 上線， 標誌著 NeoX 生態正式形成，涵蓋 NFT、遊戲、藝術、元宇宙場景等多領域。

NeoX Coin（NXC）

NXC 是 NeoX 生態的核心實用性代幣，用於未來的應用場景、 使用者互動、獎勵機制，以及 NeoX 遊戲與平台的功能啟動。

NFT 系列

NeoX 包含多個具代表性的藝術 NFT 系列，每個系列代表不同創作方向：

  • NeoX Timekeeper Edition — 8 張限量 NFT，主題圍繞時間與感知。
  • NeoX Van Gogh Visions — 以梵高風格重塑的藝術系列。
  • NeoX Abstract Realms — 高概念抽象藝術創作。
  • NeoCoin Genesis Edition (G1) — NeoX 早期基礎藝術系列。

NeoX Catch 遊戲

NeoX Catch 是一款 HTML5 網頁遊戲，玩家需收集金幣、避開障礙並通過不同關卡。 遊戲特色包括：

  • 多關卡模式
  • 動畫開場系統
  • 排行榜與 Top 3 機制
  • 手機優化操作
  • 完全融入 NeoX 視覺風格

元宇宙展覽館

NeoX 在 OnCyber 及 3D 虛擬空間內設有展區， 展示 NFT、3D Avatar、AI 動態影像與品牌創意內容，形成沉浸式藝術體驗。

技術架構

NeoX 技術堆疊包括：
• Base L2 主網
• Thirdweb 智能合約部署
• IPFS / Lighthouse 儲存
• VRM / glTF 虛擬角色
• HTML5 + Canvas 遊戲引擎
• 全平台（尤其 iOS）響應式設計

未來發展方向

NeoX 計劃擴展更多虛擬展館、推出新 NFT 系列、製作 AI 微電影、 增強遊戲體驗，以及打通更多跨平台整合功能。
