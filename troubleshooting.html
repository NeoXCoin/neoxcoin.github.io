Troubleshooting Guide

If you're experiencing issues when using NeoX Coin with MetaMask or purchasing NXC, here are some common fixes:

  • Problem: Clicking Buy via MetaMask doesn't open the app directly
    Solution: On some iPhones, Safari may not redirect properly. Try using Chrome or MetaMask's in-app browser (open MetaMask app → Browser tab → visit https://neoxcoin.github.io).

  • Problem: Add to MetaMask button doesn't work
    Solution: Make sure MetaMask is installed and your browser supports Web3 interactions (e.g., MetaMask mobile, desktop with extension).

  • Problem: You can't see the token after adding it
    Solution: Check your MetaMask network is set to Sepolia (testnet), and refresh the token list manually.
