01 · 品牌總覽（Brand Overview）

NeoX Coin (NXC) 是部署於 Base Mainnet 的新一代數位資產， 目標是將 NFT、數位藝術、互動遊戲以及類元宇宙展覽等 不同形式的 Web3 體驗整合到同一個品牌之下。

NXC 被設計為一枚 創意導向的功能型代幣， 而非投機產品；它用來串連 NeoX 的 NFT 收藏系列、 瀏覽器小遊戲、虛擬展覽與未來的數位資產體驗， 讓使用者可以在一個清晰、一致的品牌之中接觸鏈上文化。

NeoX 著重於 清晰度、創造力與長期品牌身份， 並在所有平台上維持統一的視覺與敘事風格。

02 · 願景與使命（Mission & Vision）

願景 Vision

建立一個容易接觸的 Web3 生態，讓藝術、遊戲、 數位身份與社群，可以在同一個品牌底下自然連結， 並以 NXC 作為核心媒介。

使命 Mission

  • 提供一枚簡單但功能強大的多平台實用代幣。
  • 為創作者提供開放、易用的工具與展示空間。
  • 打造兼具藝術性與互動性的遊戲與視覺體驗。
  • 讓更多日常使用者可以輕鬆接觸鏈上文化。

03 · NeoX 代表什麼？（What NeoX Represents）

NeoX 結合了 高質感數位品牌形象、 精緻視覺藝術，以及 以實用性為核心的設計。

核心象徵包括：

  • 金色（Gold） → 價值、真實與信任。
  • NeoX 藍（NeoX Blue） → 科技、創新與未來感。

這種金色與藍色的雙色語言，貫穿 NeoX 所有視覺， 包括 NFT、Logo、遊戲介面以及元宇宙展覽空間。

04 · 代幣總覽（NXC）

NeoX Coin (NXC) 是部署於 Base Mainnet 的 ERC-20 功能型代幣， 擔任整個 NeoX 生態系的核心資產，負責開啟各種數位體驗、 權限與未來的功能模組。

Name NeoX Coin
Ticker NXC
Network Base Mainnet（以太坊 Layer-2）
Standard ERC-20
Contract Address 0x9C17fDa33CBEe4aE3768252Fd62156034F42B40B

目前與未來的實用性（Utility）：

  • 用於存取 NeoX 生態中的特定功能與限定內容。
  • 與官方 NFT 收藏系列及瀏覽器小遊戲進行整合。
  • 未來可應用於數位道具、遊戲內互動與市場流程。
  • 後續開發階段將加入社群驅動的額外實用功能。

備註：NXC 為以實用性為主的功能型代幣，並非投資產品。

07 · NFT 生態系

NeoX 已推出多個精心策劃的 NFT 收藏系列， 在視覺風格與故事層面都保持一致。

  • NeoX Timekeeper Edition – 在 Base Mainnet 上發行的 8 枚官方 NFT。
  • NeoX Van Gogh Visions – 向經典畫風致敬並加入未來感的系列作品。
  • NeoX Abstract Realms – 以抽象數位藝術為主的實驗系系列。
  • NeoX Genesis & Genesis Edition – 奠定 NeoX 品牌視覺語言的早期作品。

部分系列可於 OpenSea 之官方帳戶下瀏覽與交易。

08 · 遊戲與元宇宙體驗

NeoX 正持續開發多種結合藝術與互動的 Web3 體驗，包括：

  • NeoX Catch the Coins – HTML5 小遊戲，透過接幣機制與分數系統增加互動性。
  • NeoX Metaverse Exhibition – 透過 OnCyber 建立的虛擬展覽空間。
  • 未來規劃包括更多關卡、排行榜機制，以及將 NFT 與 NXC 更深入地整合到遊戲與互動體驗之中。
