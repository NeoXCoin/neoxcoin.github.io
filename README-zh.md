## 🚀 GitHub Pages 部署方式

1. 進入 GitHub 專案 → **Settings → Pages**
2. 選擇：Branch = `main`，資料夾 = `/ (root)`
3. 儲存後稍等，網站即部署於：`https://<你的帳號>.github.io`
