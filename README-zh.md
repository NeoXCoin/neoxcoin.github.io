## 🚀 GitHub Pages 部署方式

1. 前往 GitHub 專案 → **Settings → Pages**  
2. 設定：**Branch = `main`**，資料夾選擇 **`/(root)`**  
3. 儲存後稍等片刻，即可於以下網址查看網站：  
   🔗 https://neoxcoin.github.io
