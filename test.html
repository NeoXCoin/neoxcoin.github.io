NeoX Coin (NXC) – Whitepaper

NeoX Coin (NXC) is a community-driven utility token on Base Mainnet powering the NeoX ecosystem of NFTs, digital art, casual gaming, and metaverse exhibitions.

Version 1.2 · Last updated: 11 November 2025

1. Abstract

NeoX Coin (NXC) is an ERC-20 token deployed on Base Mainnet and designed as the core utility token for the NeoX ecosystem. Instead of focusing purely on speculation, NeoX emphasizes creativity and playful Web3 experiences that connect NFTs, interactive content, and virtual exhibitions.

2. Vision & Motivation

“Web3 doesn’t have to be only about price charts – it can also be about art, stories, and experiences.”

  • Art-first: NFTs represent stories and aesthetics, not just assets.
  • Fair launch: No presale or private sale. Equal access for the community.
  • Multi-format experiences: Includes illustration, animation, music, games, and exhibitions.
  • Low fees & accessibility: Built on Base Mainnet for low gas and global access.

3. Technology Stack

  • Blockchain: Base Mainnet (Ethereum Layer 2)
  • Token Standard: ERC-20
  • Contract: 0x9C17fDa33CBEe4aE3768252Fd62156034F42B40B
  • Explorer: View on BaseScan

NeoX uses thirdweb’s standardized smart-contract framework to reduce deployment risk while ensuring simplicity and security.

4. Token Design

  • No rebasing, reflection, or complex mechanics.
  • No presale/private sale: Fair launch model.
  • Fixed supply: Set at deployment; no further minting planned.
  • Admin control: Limited to thirdweb’s standard safety parameters.
