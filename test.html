Fund your wallet with ETH on Base

Base Network requires ETH to cover transaction gas fees and purchase NXC tokens.

Option A (Recommended): Securely withdraw ETH directly via the Base Network from popular exchanges like Coinbase or Binance.

Option B: Bridge your Layer-1 ETH from Ethereum Mainnet using the Official Base Bridge.

準備 Base 網路上的 ETH

Base 網路需要少量的 ETH 來支付礦工費（Gas 費）及兌換 NXC 代幣。

方法 A (最省錢推薦): 直接從交易所（如 Binance 或 Coinbase）提現 ETH 時，網路選擇 Base 網路 轉入您的錢包。

方法 B: 使用 官方 Base 跨鏈橋，將以太坊主網的 ETH 跨鏈至 Base 網路。
