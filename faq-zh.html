常見問題
問：NeoX 是詐騙嗎？
答：當然不是。NeoX 是一個具透明度的教育型 Web3 專案。所有智能合約皆為開源且可驗證，我們的資產儲存在 GitHub 及 IPFS 等可信平台。我們不承諾不切實際的收益，也不向使用者募資 — NeoX 是一項公開、由社群驅動的實驗性項目。
問：NeoX 的宗旨是什麼？
答：NeoX 是一個以 Web3、設計及數位價值探索為核心的去中心化代幣專案，涵蓋 NFT 系列、教育資源、創新實驗等元素。
問：為什麼我可以信任這個專案？
答：我們的智能合約、元資料與 IPFS 資源皆完全公開，並可於 GitHub 查閱。社群平台亦提供項目資訊，歡迎大家自行研究（DYOR）以確認其可信度。
問：我要如何購買或測試 NeoX 代幣？
答：請造訪「如何購買」頁面查看詳細教學。我們支援測試網與 MetaMask 整合。
問：這是官方網站嗎？
答：是的，這裡是 NeoX 官方網站。如欲關注更多動態，請追蹤我們的官方 Instagram：@neox__official。
