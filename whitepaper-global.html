1. Abstract

2. Vision & Motivation

The NeoX project was created with a simple idea: Web3 should feel artistic, playful, and human ‚Äì not only financial. NeoX focuses on storytelling through NFTs, interactive mini-games, and immersive galleries, while using NXC as the connective layer for value and participation.

  • Art-first: Highlight digital art and curation instead of pure speculation.

  • Global accessibility: Leverage Base Mainnet for cost-effective transactions and multi-language support via this global edition.

4. Token Design

NXC is intended to be a straightforward, transparent token with no complex rebasing or reflection mechanisms. It is designed primarily for utility within the NeoX ecosystem rather than as a high-risk financial instrument.

  • Presale / Private Sale: None ‚Äì fair community launch.

  • Admin controls: Limited to standard configuration and security parameters provided by thirdweb.
