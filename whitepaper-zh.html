NeoX Coin (NXC) – 白皮書

NeoX Coin (NXC) 是 NeoX 元宇宙生態的核心代幣，串連 NFT 藝術、遊戲互動與沉浸式 Web3 體驗。

NeoX 元宇宙官方說明文件 · 版本 1.0 · 最後更新：2025 年 11 月

1. 簡介

NeoX Coin (NXC) 建立在 Base Mainnet 之上，目標不是做一枚只用來炒作的代幣， 而是一個陪住創作者、玩家、收藏家一齊成長的「生態核心」。 透過 NXC，NeoX 希望將數碼藝術、輕量遊戲、虛擬展覽空間，慢慢組合成一個有故事、有靈魂的 Web3 小宇宙。

本白皮書以清晰易讀的方式，介紹 NeoX 的理念、技術基礎、代幣設計，以及 NXC 在整個 NeoX 元宇宙中的實際角色。

2. 願景與理念

NeoX 一開始的想法好簡單：
「Web3 不應該只有 K 線圖和暴漲暴跌，它亦可以好藝術、好好玩，甚至好溫度。」

圍繞這個想法，NeoX 有幾個核心理念：

  • 藝術優先： 先做好內容與美學，再談其他。NFT 不只是一張圖，而是一個故事、一個場景、一種情緒。
  • 公平參與： 無預售、無私募，希望盡量減少「內部人優勢」，用更平均的方式去建立社群。
  • 多形態體驗： 不只做一種項目，而是用 NeoX 品牌去承載圖像、動畫、音樂、簡單遊戲及虛擬展覽空間。
  • 低成本、易接觸： 選擇 Base Mainnet，令 gas 費較低，方便更多新手願意嘗試接觸 Web3。

3. 技術基礎

NeoX Coin (NXC) 是一枚遵從 ERC-20 標準的代幣，部署於 Base Mainnet，並透過 thirdweb 的標準化合約進行部署與管理。

  • 區塊鏈： Base Mainnet（以太坊 Layer 2）
  • 代幣標準： ERC-20
  • 合約地址： 0x9C17fDa33CBEe4aE3768252Fd62156034F42B40B
  • 區塊鏈瀏覽器（BaseScan）： 按此於 BaseScan 查看

使用 thirdweb 的優勢，是可以依賴一套被廣泛使用的智能合約框架，簡化部署流程，同時保留基本配置及安全機制。

4. 代幣設計

NXC 的設計方向偏向「簡單、透明、易理解」。
沒有複雜的 Rebase、反射機制，亦不是一枚高風險實驗型代幣，而是為 NeoX 生態而生的實用型 Token。

  • 預售 / 私募： 無。NXC 以公平社群形式啟動。
  • 供應量： 於部署時設定，之後不預期再額外增發。
  • 合約控制： 僅限 thirdweb 提供的標準管理與安全相關設定。

關於技術層面最準確的資訊，以 BaseScan 上之合約資料為最終依據。

5. NXC 在 NeoX 生態中的應用

NXC 被視為 NeoX 元宇宙的「通用代幣」，未來可在不同體驗中扮演多種角色。以下為構想方向（實際落地會分階段推出）：

  • NFT 生態應用：
    未來部分 NFT 發行、限量空投、鑄造折扣或收藏家回饋，有機會結合 NXC 作為支付或獎勵工具。
    相關系列包括（但不限於）：NeoX Timekeeper Edition、NeoX Van Gogh Visions、NeoX Abstract Realms 等。
  • 遊戲化體驗：
    在例如「NeoX Catch the Coins」這類輕量遊戲中，NXC 未來有機會用於解鎖皮膚、特別道具、活動獎勵或排行榜相關互動（具體形式將視開發進度調整）。
  • 虛擬展覽與互動空間：
    NeoX 的 3D 或沉浸式展覽空間中，日後有可能出現以 NXC 解鎖的特別展區、導覽模式或互動裝置。
  • 社群與活動：
    NXC 亦可以應用於社群任務、創作挑戰、抽獎活動或合作企劃，作為一種「參與度」的象徵。

以上用途為方向性規劃，實際功能將按技術、法規與社群狀況逐步推出，並以官方公告為準。

6. 生態組成

NeoX 生態目前包含多個不同元素，並會持續擴展：

  • NFT 系列：
    包括 NeoX Timekeeper Edition、NeoX Van Gogh Visions、NeoX Abstract Realms 等，以不同主題探索藝術風格與故事。
  • 互動式網頁體驗：
    例如圍繞 NeoX Coin 的小遊戲、動畫式開場畫面、動態網站設計等，令訪客不只是「看」，而是可以「玩」。
  • 元宇宙展覽空間：
    利用虛擬展館呈現 NeoX 作品，讓收藏家與訪客可以用更沉浸的方式認識整個世界觀。

7. 發展路線圖（高層次）

NeoX 的路線圖保持一定彈性，因為創作靈感與技術可能隨時間演變。以下為簡化版階段規劃：

  • 第一階段：基礎建立
    在 Base Mainnet 部署 NXC，推出首批 NFT 系列，搭建官方網站與品牌形象。
  • 第二階段：體驗擴展
    持續優化 NeoX 主頁與互動體驗，完善 NeoX 遊戲概念，強化視覺統一感與故事線。
  • 第三階段：實用功能深化
    逐步嘗試將 NXC 與 NFT 鑄造、獎勵系統或特別活動結合，視實際情況分階段推出。
  • 第四階段：社群與合作
    與更多創作者、設計師或志同道合的項目合作，在 NeoX 品牌下拓展更多藝術與遊戲內容。

8. 風險與免責聲明

任何涉及區塊鏈與加密資產的行為，都伴隨一定程度的風險。
NXC 並非傳統金融產品，亦不應被視為保證收益的投資工具。

  • 代幣價格可能大幅波動，亦有可能流通性不足。
  • 智能合約雖然使用標準化方案，仍存在潛在技術風險。
  • 不同地區對加密資產的監管規定可能隨時間改變。
